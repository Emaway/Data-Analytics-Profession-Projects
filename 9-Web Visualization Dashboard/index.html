Summary: Latitude Vs. X

The purpose of this project is to develop Web Viaualization Dashboard

The main objective of weather analysis was to explore the relationship between temperature, humidity, cloudiness and wind speed with respect to distance from the equator.

To perform weather analysis, we first pulled data from OpenWeatherMap API on 500+ cities across the world selected randomly. A weather check was performed on each of these cities using a series of successive API calls from OpenWeatherMap. Then all different points of interest were plotted against latitude using Matplotlib.

This part provides the source data and visualizations created as part of analysis, as well as explanations and descriptions of any trends and correlations witnessed.
